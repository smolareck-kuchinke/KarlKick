# KarlKick

KarlKick ist eine moderne Fußball-Tippspiel-App.

Dieses Projekt entsteht schrittweise mithilfe von AI Coding (Claude Code).

## Technologie

- React (Frontend)
- Spring Boot
- Maven
- Java 21
- H2 Database
